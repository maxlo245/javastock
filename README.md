# Javastock

Projet de gestion de stocks pour une course (Java, Swing, PostgreSQL).

## Structure
- `src/JavaStocks/` : code source Java
- `bin/JavaStocks/` : fichiers compilés
- `pom.xml` : configuration Maven

## Compilation

Compilez avec Maven :
```
mvn clean package
```

Ou en ligne de commande :
```
cd src
javac JavaStocks/MainMenu.java
java JavaStocks.MainMenu
```

## Fonctionnalités
- Menu graphique principal (Swing)
- Gestion des articles, coureurs, types d'épreuve, réservations
- Connexion à PostgreSQL (à compléter)
