🚀 COMMENT UTILISER
═══════════════════════════════════════════════════════════════
ÉTAPE 1: Lancer l'application
   → Double-cliquez sur run.bat
   → Le Menu Principal s'ouvre
ÉTAPE 2: Sélectionner une catégorie
   → Cliquez sur "Gestion des articles" (ou autre)
   → Cliquez sur [Valider]
ÉTAPE 3: Remplir le formulaire
   → Complétez tous les champs obligatoires
   → Les champs vides affichent une erreur
ÉTAPE 4: Envoyer
   → Cliquez sur le bouton vert [ENVOYER]
   → Message de confirmation s'affiche
   → Données sauvegardées en base PostgreSQL
ÉTAPE 5: Continuer ou Fermer
   → Le formulaire se réinitialise automatiquement
   → Vous pouvez ajouter un autre élément
   → Ou cliquez sur [Fermer] pour revenir au menu
